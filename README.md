# RetiHtmlProxy

Add to your browser access to Reticulum Html. Acces to link with Hash, like http://c6923743983924061686da9a2061dcb4

# Install

pip install rns

pip install requests

you can also check the video how to install and use it https://youtu.be/MctX5SwKRMQ
# Run

python RetiHtmlProxy.py 

## In the browser
go to setting network proxy, manual proxy. http: 127.0.0.1 8080 (only activate for http)

in the url bar enter http://c6923743983924061686da9a2061dcb4

you will appear on Wikipedi English with no picture.

or here http://d91a4e029d62fc64ca7d8f54f52efd11
for Reticulum Website.

compatible with rserver  https://github.com/guyroyse/rserver
and rwebserver https://github.com/cotteux/rwebserver
